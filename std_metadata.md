# Standard Metadata

## `id`
A unique identifier for the story. Optional. Defaults to the story name.

## `required_verbs`
A list of verbs that must be registered for the story to be valid. This should be checked by a pre-processor.

If the value is `"*"`, it means all mentioned verbs must be registered.

## `zoh_version`

State the zoh spec version the story is written for. Optional. Defaults to the latest version.
